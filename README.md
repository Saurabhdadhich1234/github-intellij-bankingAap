# Banking Application

This Java Spring Boot project is a basic banking application designed for development in IntelliJ IDEA. It features backend services providing RESTful APIs for common banking operations, including account management and transaction processing. The app also includes utilities for reading files to support data operations.

## Features
- Account and transaction management
- REST API endpoints for banking services
- File reading capability for data import/export
- Easy setup with IntelliJ IDEA

## Getting Started
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Build and run the Spring Boot application
4. Use API endpoints to interact with banking services

## Requirements
- Java 11+
- Spring Boot
- Maven or Gradle
